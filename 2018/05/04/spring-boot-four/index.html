随机数

RandomValuePropertySource：配置文件中可以使用随机数

例如${random.value}、${random.int}、random.long、${random.int(10)}、${random.int[1024,65536]}

占位符

  • 可以在配置文件中引用前面配置过的属性（优先级前面配置过的这里都能用）
  • 占位符获取之前的值，如果没有可以使用:指定默认值
